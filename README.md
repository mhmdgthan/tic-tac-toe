# Tic Tac Toe

Game Tic Tac Toe sederhana yang dibuat menggunakan HTML, CSS, dan JavaScript.

## Fitur

- Permainan 2 pemain bergantian (X dan O)
- Deteksi kemenangan untuk semua baris, kolom, dan diagonal
- Deteksi seri
- Menyorot kombinasi pemenang
- Tombol Main Lagi untuk mengulang permainan

## Cara Bermain

1. Buka `index.html` di browser
2. Pemain X jalan duluan
3. Bergantian klik kotak
4. Pemain pertama yang mendapat 3 berturut-turut menang
5. Klik **Play Again** untuk mulai ulang

## Teknologi

- HTML
- CSS
- JavaScript (Vanilla)

## Demo
